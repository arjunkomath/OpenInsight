# OpenInsight

> Query your database with natural language. Claude-like chat interface for business intelligence.

> [!WARNING]  
> **Lot of the code was written by AI (Claude)**, this is an experiment.

## Quick Start

OpenInsight runs on [Bun](https://bun.sh/) (>=1.3). Install Bun first if you don't have it.

```bash
# 1. Install
bun add -g openinsight

# 2. Set your OpenRouter API key
export OPENROUTER_KEY=sk-...

# 3. Run
openinsight
```
